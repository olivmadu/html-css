Usando links externos

É um link que aponta para um outro site (que não é o meu). Os links são a ligação entre uma página e outra. EX: acessar o repositório público no GitHub.

Você também pode acessar o canal gratuito no Youtube.

Usando links internos

Vai fazer a relação com outra página que está dentro do próprio servidor e diz respeito à própria página. Quando faz o link interno, não precisa indicar a URL inteira.
